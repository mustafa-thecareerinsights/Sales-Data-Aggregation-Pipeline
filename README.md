# Sales Data Aggregation Pipeline

## Objective

Build a Python pipeline to process sales data and generate summary statistics.

## Features

- Read sales data from CSV
- Validate records
- Handle missing data
- Calculate total sales
- Group sales by category
- Find top-selling product
- Export summary CSV
- Exception handling

## Input

data/sales_data.csv

## Output

data/sales_summary.csv

## Run

```bash
python src/sales_pipeline.py
```

## Sample Results

- Total Sales
- Sales by Category
- Top Selling Product
